# App Privacy Answers

Use this as the basis for App Store Connect's App Privacy section.

## Does this app collect data?

No.

## Data used locally

BarLens reads local running app metadata and visible menu bar window metadata to show the interface. It stores per-app visibility rules locally in the user's Application Support directory or app container.

## Tracking

No tracking.

## Third-party data sharing

None.

## Analytics

None.

## Advertising

None.

## Notes

The app does not include third-party SDKs and does not make network requests except when the user explicitly opens the Privacy Policy or Support links from the menu.
