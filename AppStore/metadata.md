## Description

BarLens is a lightweight macOS utility for inspecting the menu bar and organizing the apps that may provide status items.

It scans visible menu bar window metadata through public macOS APIs, lists likely menu bar providers, and lets you keep a local rule for each app: always show, hide when inactive, keep hidden, or observe only. You can pin important providers, open them, or quit providers you no longer need.

BarLens is designed to be transparent about macOS limits. Apple does not provide a public API that lets one app freely move or hide every other app's menu bar icon, so BarLens focuses on detection, local organization, and macOS-permitted controls instead of private system behavior.
