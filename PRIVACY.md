# Privacy Policy

BarLens runs locally on macOS.

## Data used

- Running application names, bundle identifiers, process identifiers, and icons from `NSWorkspace`.
- Visible top menu bar window metadata from `CGWindowList`.
- Per-app visibility preferences saved locally under Application Support.

## Data not collected

- No analytics.
- No telemetry.
- No network requests from the app.
- No uploaded menu bar data.

GitHub release downloads are served by GitHub, so GitHub's normal download logging may apply.
